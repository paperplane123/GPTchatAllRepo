# 快递一句话下单（kuaidi-oneclick）

基于快递100“上门取件 API（线下支付）”的零依赖 Python CLI。寄件人只配置一次，以后用一句中文描述收件人、地址、取件时间和物品即可生成订单。

## 安全原则

- 默认仅预览，**不会真实下单**；必须显式追加 `--submit`。
- `KUAIDI100_KEY`、`KUAIDI100_SECRET` 只从环境变量读取，不写入配置或仓库。
- 同一天内容完全相同的订单默认使用相同 `thirdOrderId`，减少重复提交风险；确实需要重复寄件时加 `--new-order`。
- 生产密钥一旦贴到聊天、日志或截图中，应立即在平台重置。

## 环境要求

Python 3.10+，不需要安装第三方包。

## 1. 初始化

```bash
python3 ship.py --init
```

按提示填写：

- 寄件人姓名、手机号、完整地址
- 公网可访问的 `callBackUrl`

默认配置保存在：

```text
~/.kuaidi-oneclick/config.json
```

> 快递100文档将 `callBackUrl` 列为下单必填项，本地 `localhost` 地址无法接收平台回调。

## 2. 配置凭证

仅在当前终端会话设置：

```bash
export KUAIDI100_KEY='你的key'
export KUAIDI100_SECRET='你的secret'
```

不要把真实值写入 `.env` 后提交到 GitHub。需要长期使用时，可放入 macOS 钥匙串、1Password CLI 或部署环境的 Secret 管理功能。

## 3. 一句话预览

```bash
python3 ship.py '顺丰到付；明天09:00-11:00；收件人张三；手机13800138000；地址广东省深圳市南山区科技园某路1号；物品文件；重量1kg'
```

程序会输出脱敏后的请求参数，并明确提示当前没有调用接口。

推荐句式：

```text
顺丰到付；明天09:00-11:00；收件人张三；手机13800138000；地址完整收件地址；物品文件；重量1kg；备注易碎
```

支持的快递公司：顺丰、京东、德邦、圆通、中通、顺丰快运、顺心捷达、跨越、EMS。

支持重量单位：`kg`、公斤、千克、斤（自动换算为 kg）。未写付款方式时默认为寄付；未写物品或重量时使用配置中的默认值。

## 4. 真正提交

测试环境：

```bash
python3 ship.py '顺丰到付；明天09:00-11:00；收件人张三；手机13800138000；地址广东省深圳市南山区科技园某路1号；物品文件；重量1kg' --submit --test
```

正式环境：

```bash
python3 ship.py '顺丰到付；明天09:00-11:00；收件人张三；手机13800138000；地址广东省深圳市南山区科技园某路1号；物品文件；重量1kg' --submit
```

正式环境提交后，快递公司可能安排上门揽件。请先预览并核对姓名、电话、地址、付款方式和预约时间。

## 常见错误

- `503 验证签名失败`：检查 key/secret 是否匹配；程序会按 `MD5(param + t + key + secret)` 生成 32 位大写签名。
- `600/601`：key 不合法、过期或账号无可用单量。
- `400 参数错误`：检查快递公司、地址、手机号、必填字段和接口权限。
- 回调收不到：确认 `callback_url` 是公网 HTTPS 地址，并能处理 `application/x-www-form-urlencoded`。
