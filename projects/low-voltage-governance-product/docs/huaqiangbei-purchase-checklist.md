# 华强北询价与装配清单

## 一句话需求

> 做一台 15.6 英寸触摸屏手提箱式 Windows 一体机，内置 32GB+1TB 高性能迷你主机、隔离 RS485、实体状态灯/按键/旋钮，能运行本地 Web 系统和 ETAP；主机箱内不接裸露 220/380V，真实三相采集放在独立导轨箱。

## 1. 先询三家，不接受当场换规格

向每家供应商发送同一份规格，要求分别报价：

- 整机价格；
- 交期；
- 主机、屏幕、内存和 SSD 的具体品牌型号；
- 机箱尺寸、材质和加工方式；
- 保修期限；
- 是否提供系统镜像、接线图和 BOM；
- 是否支持 8 小时老化测试；
- 是否可开票。

未经确认不得把：

- 8845HS/8945HS 换成 N100/N150；
- 32GB 换成单条 16GB；
- TLC SSD 换成无品牌 QLC；
- IPS 触摸屏换成 TN 或非全贴合屏；
- 隔离 RS485 换成普通 CH340 裸板；
- 双网口换成单百兆网口。

## 2. 主机验货

- CPU-Z 核对 CPU 型号、核心数和内存双通道；
- CrystalDiskInfo 核对 SSD 品牌、容量、通电时间和健康度；
- 检查 Windows 激活方式；
- 双网口同时连接测试；
- USB-A、USB-C、HDMI、DP 全部测试；
- BIOS 恢复来电自启动；
- 设置温度异常与风扇策略；
- 运行 30 分钟 CPU 压力测试，无蓝屏、重启和异常降频；
- 至少保留一个可直接插加密锁的 USB-A 口。

## 3. 屏幕验货

- 1920×1080 原生分辨率；
- Windows 下 10 点触控；
- 全屏检查亮点、坏点、漏光；
- 触控边缘校准；
- HDMI 与 USB 触控线可拆换；
- 最低亮度下无明显闪烁；
- 固定后屏幕不能晃动或压迫排线；
- 预留外接 HDMI OUT 给 ETAP 大屏。

## 4. RS485 验货

- 外壳或板卡明确标注隔离；
- 查看芯片和保护器件，不接受仅口头声称隔离；
- 端子为 A、B、GND，并有防松结构；
- Windows 11 驱动正常；
- 9600/19200/38400 波特率测试；
- 连续轮询 Modbus 模拟器 2 小时无掉线；
- 拔线后软件必须显示通信中断，不继续显示伪造实时值。

## 5. 结构与安全验货

- 市电入口带总开关和保险丝；
- 使用漏保插头；
- 原厂电源适配器固定牢靠且可更换；
- 无裸露市电端子；
- 内部 AC 与弱电线束分开；
- 所有线束做冷压、热缩、标签和应力释放；
- 风扇进出风方向明确，防尘网可拆；
- 主机、Hub、适配器和小板均可独立拆换；
- 航空箱关闭后不能挤压屏幕和按键；
- 总重量目标不超过 8kg。

## 6. 交付时必须拿走

- 完整整机；
- 电源线和所有外接线；
- 备用保险丝、端子和至少一只备用按键；
- Windows 管理员账号；
- BIOS 密码（若设置）；
- 系统恢复镜像；
- BOM 与每项实际型号；
- 内部接线图；
- 机箱 CAD/DXF 开孔文件；
- 软件安装包和驱动备份；
- 老化测试记录；
- 发票和保修联系方式。

## 7. 采购决策红线

出现以下任一情况直接换供应商：

- 不肯写明主机、屏幕、内存、SSD 型号；
- 不允许验机；
- 用破解系统或无法说明授权；
- 市电接线裸露、胶布缠线、无保险；
- RS485 无隔离仍宣称“工业级”；
- 机箱封死、不能维护；
- 只提供口头保修；
- 把演示样机说成可直接安装到现场配电箱。
