# 低电压治理软硬一体机硬件选型 V0.1

> 状态：华强北可快速攒机的展示/验证样机方案，不是生产定型，也不能直接作为现场带电装置使用。

## 1. 样机定位

第一台不做定制主板，不做户外台区终端，也不把 220/380V 裸接线放进展示主机。先做一台可携带、可触控、可接 RS485 表计、可本地运行产品原型和 ETAP 的桌面一体机，用于：

- 给领导和合作方演示完整业务闭环；
- 本地运行低电压治理 Web 产品；
- 接入一只真实三相表计或 Modbus 模拟器；
- 演示监测、诊断、方案比选、工单与报告；
- 在具备 ETAP 授权时运行小规模项目和自动化接口；
- 后续拆分为主机、采集箱和云平台三层架构。

产品形态建议为 **15.6 英寸铝合金手提箱式控制台**：上盖为触摸屏，下层安装迷你主机、通信模块、状态灯和实体按键。

## 2. 推荐总方案

### 2.1 核心计算单元

推荐规格：

- x86-64 迷你主机；
- AMD Ryzen 7 8845HS / Ryzen 9 8945HS，或 Intel Core i7-12650H / i7-13700H 同级；
- 32GB DDR5；
- 1TB NVMe SSD；
- 双 2.5GbE 网口优先；
- 至少 4 个 USB-A、1 个 HDMI、1 个 USB-C/USB4；
- Windows 11 Pro；
- VESA 安装孔；
- 预留 USB-A 给 ETAP 硬件加密锁。

首选参考机型：**MINISFORUM UM890 Pro 32GB+1TB**，或华强北同规格白牌迷你主机。它具备 8 核 16 线程、双内存插槽、双 M.2、双 2.5GbE、HDMI/DP/双 USB4，足够支撑本地 Web、数据库、Modbus 采集和小规模 ETAP 演示。

不建议：

- Intel N100/N150：能跑前端，但留给 ETAP、数据库和批量计算的余量不足；
- ARM 开发板：ETAP 和 Windows 外设兼容风险更高；
- 带独立显卡的游戏主机：功耗、体积和散热成本不划算；
- 8GB/256GB 配置：安装 ETAP、项目文件和数据库后余量太小。

### 2.2 显示与触控

推荐：

- 15.6 英寸 IPS；
- 1920×1080；
- 10 点电容触控；
- HDMI 视频 + USB 触控；
- 钢化玻璃；
- 300nit 以上；
- VESA 或可拆外壳；
- Windows 免驱。

首选参考：**Waveshare 15.6inch HDMI LCD (H)** 或华强北同面板定制版本。

说明：15.6 英寸用于我们自己的产品界面，不用于直接操作复杂 ETAP 图纸。需要精细编辑 ETAP 时外接 24/27 英寸显示器。

### 2.3 实体交互与产品感

采用一个 RP2040/STM32 USB-HID 小板，不参与电气计算，只负责：

- 三色状态灯：绿色正常、黄色预警、红色低电压；
- 三个带灯按键：`开始诊断`、`生成方案`、`确认工单`；
- 一个带按压旋钮：切换问题点/方案；
- 蜂鸣器：仅演示告警，可在软件中关闭；
- 面板丝印产品名称和状态说明。

该模块即使故障，也不能影响主系统计算和数据完整性。

### 2.4 通信接口

样机首选：**隔离型 USB-RS485 转换器**。

最低要求：

- FT232RNL/FT2232 等稳定芯片；
- 2kV 以上隔离；
- TVS、ESD、浪涌保护；
- 接线端子；
- 自动收发方向控制；
- Windows 11 驱动稳定；
- 120Ω 终端电阻可配置。

首选参考：Waveshare USB TO RS485 (C)。需要同时接多只设备时换成 2 路或 8 路隔离型号。工程样机可升级为 Moxa UPort 1150I。

### 2.5 三相电量采集

展示样机分成两种模式：

#### 安全演示模式

- 使用软件回放历史曲线；
- 或使用 RP2040/ESP32 模拟 Modbus 表计寄存器；
- 旋钮改变 A/B/C 相电压、负载率和不平衡度；
- 主机通过 RS485 读取，完整演示“硬件采集—诊断—方案”的流程；
- 全部使用 5V/12V 安全低压。

#### 现场验证模式

外接独立 DIN 导轨采集箱，推荐：

- 安科瑞 ADL400N-CT 或 ADL400；
- 三相四线 220/380V；
- RS485 Modbus-RTU；
- 外置开口 CT 优先；
- 独立微型断路器、熔断器、端子和防护外壳；
- 由具备资质的电工完成接线。

主控制台只接隔离后的 RS485，不在箱体内部暴露市电端子。

说明：普通计量表可以支撑相电压、电流、功率和能量演示，但若正式产品要做电压暂降、谐波和电能质量分析，应另选专业电能质量监测装置，不能拿普通电能表代替。

### 2.6 网络

样机默认：

- 有线千兆/2.5GbE；
- Wi-Fi 6；
- 手机热点作为备用。

可选 4G：

- 展示阶段使用华强北成品 4G 工业路由器；
- 现场 PoC 再升级为带双 SIM、看门狗、VPN 和远程管理的工业路由器；
- 不建议第一台就购买高价 5G 工业路由器。

### 2.7 供电与续航

展示样机优先使用各设备原厂认证电源适配器，统一接入箱内带保险的 AC 插座模块。不要在第一台样机中自行设计 220V AC-DC 电源板。

推荐：

- 带保险丝、总开关和 EMI 滤波的 IEC 电源入口；
- 漏电保护插头；
- 迷你主机原装 19V 适配器；
- 屏幕原装 12V 适配器；
- 5V USB 隔离电源给交互板；
- 选配 19V/12V 双输出 DC UPS，目标续航 10—20 分钟；
- 断电后软件自动保存并安全关机。

工程版再统一为 24V DIN 导轨电源与隔离 DC-DC。

### 2.8 机箱结构

推荐华强北加工方式：

- 16—17 英寸铝合金手提航空箱；
- 上盖嵌 15.6 英寸屏；
- 下层激光切割 2mm 铝面板；
- 主机隐藏安装并保留可拆检修盖；
- 两侧强制进出风，带可拆防尘网；
- 背部接口：AC、LAN1、LAN2、RS485、USB、HDMI OUT；
- 前面板：三色灯、三个按键、旋钮、总电源；
- 所有市电接线使用冷压端子、热缩管和防触碰护罩；
- 主机与采集箱使用可插拔航空插头或工业接线端子。

外观不要做成“电脑加屏幕”，要做成一台可携带的业务终端。

### 预算结论

- **只做安全演示机：约 5500—7500 元**；
- **加真实三相采集箱：约 6500—9000 元**；
- **换工业主机、Moxa、工业路由和 24V 电源：约 12000—18000 元**。

华强北第一轮采购预算建议报 **8000 元封顶**，目标在 6500 元左右完成。

## 4. 明确不采购的东西

第一版不要买：

- Jetson 独立 AI 盒子；
- 独立显卡；
- 5G 工业路由器；
- PLC；
- 定制 PCB；
- 户外 IP65 机柜；
- 高精度电能质量分析仪；
- 大容量储能电池；
- 真实柔直装置或构网型储能硬件。

这些都不影响两周原型验证，采购后只会拖慢集成。

## 5. 软件部署建议

主机安装：

- Windows 11 Pro；
- Chrome/Edge kiosk 模式；
- 低电压治理产品前端与本地服务；
- SQLite/PostgreSQL 二选一；
- Python 3.11/3.13；
- Modbus 采集服务；
- ETAP 及 etapPy/etapAPI（具备授权时）；
- 开机自启动、应用看门狗、日志轮转；
- 本地演示数据一键复位。

屏幕默认只显示业务产品。ETAP 在后台或外接显示器中运行，不暴露给普通用户。

## 6. 华强北验收条件

交货时必须现场验收：

1. 冷启动 60 秒内进入产品首页；
2. 连续运行 8 小时无死机和过热降频；
3. 触摸、按钮、旋钮和状态灯全部正常；
4. RS485 连续采集 2 小时无异常掉线；
5. 断开 RS485 后软件有明确告警，不假造数据；
6. 断电前自动保存，UPS 条件下能安全关机；
7. LAN、Wi-Fi、USB、HDMI OUT 均可用；
8. 机箱内无裸露市电端子，线缆有标签和应力释放；
9. 可外接显示器运行 ETAP；
10. 提供系统镜像、BOM、接线图和钥匙/备件。

## 7. 第二阶段升级路线

V0.1 验证通过后再决定：

- 迷你主机改为无风扇工业 PC；
- 采集箱加入专业电能质量装置；
- RS485 改为多路隔离或串口服务器；
- 增加 4G/5G、VPN 和远程运维；
- 统一 24V 工业供电；
- 增加 TPM、磁盘加密和白名单；
- 完成 EMC、安规、温升和环境测试；
- 根据部署位置改为壁挂、机架或户外箱体。
