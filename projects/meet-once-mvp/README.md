# 见一面（Meet Once）MVP

一个“反滑动”的 AI 约会产品原型：用户不浏览无限信息流，而是完成简短 AI 访谈；系统每周只推荐一位候选人，双方均愿意见面后，再协调用餐或咖啡时间。

## MVP 验证目标

1. 用户是否愿意用 3 分钟完成 AI 访谈，而不是填写长表单。
2. “每周一个候选人”是否比无限滑动更有期待感和信任感。
3. 用户是否愿意在双方互选后，直接进入线下见面安排。
4. 用户是否愿意为一次被成功安排的见面付费，而不是为浏览权限订阅。

## 当前实现

- 移动端优先的单页网页原型。
- AI 访谈式建档流程。
- 每周唯一候选人展示。
- 双方同意见面的状态模拟。
- 时间和地点选择。
- 约会确认卡片。
- 浏览器 `localStorage` 保存当前进度。

当前版本不包含真实账号、后端、支付、地图、短信和大模型接口，所有候选人与匹配结果均为演示数据。

## 运行方式

直接用浏览器打开 `index.html` 即可。

也可以在本目录运行任意静态服务器，例如：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`。
